# MovieLens Data Analysis Dashboard

This Streamlit dashboard analyzes the MovieLens 200k dataset to answer questions about movie ratings, genres, and viewer demographics.

## Setup Instructions

1. Make sure you have Python installed (version 3.7 or higher)
2. Clone or download this project folder
3. Place the `movie_ratings.csv` file in a `data` folder within the project directory
4. Install the required packages:

```bash
pip install -r requirements.txt
